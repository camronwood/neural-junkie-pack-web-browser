# Web browser workspace

Use the HTML browser workbench when editing `.html` files in this workspace.

1. Enable the **Web browser** pack in **Domain packs** (⌘⇧K).
2. Open an HTML file — it opens in split editor + live preview.
3. Use **Workspace file** mode for static HTML served by the hub preview API.
4. Use **Dev server URL** mode (e.g. `http://localhost:5173`) when running Vite/Next locally.
5. Save and click **Reload preview** to refresh after edits.

Ask **@WebBrowserExpert** for layout, CSS, accessibility, or fetch_url checks on external references.
