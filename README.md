# Neural Junkie — Web browser pack

Official domain pack for [Neural Junkie](https://github.com/camronwood/neural-junkie).

Adds **WebBrowserExpert** with `fetch_url` / `web_search` MCP tools and an **HTML browser workbench** for previewing workspace HTML and local dev servers.

Requires the **software-development** pack.

Install via desktop **Domain packs** (⌘⇧K), or sideload `dist/web-browser-<version>.zip`.

## Develop

```bash
make verify
make pack-zip   # dist/web-browser-<version>.zip
```

Tag `v1.0.0` and push to publish the release zip to GitHub.
